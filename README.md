# CreateTicketAzureFunction

## Azure function to create a ticket in a freshdesk instance 

### Description

This application create ticket in a freshdesk instance using the API.

The data you need for the project are:

|Parameter|Type|
|---|---|
|userName|string|
|userEmail|string|
|Options|string|
|userText|string
